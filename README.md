# AirCanvas: Spatial Computing Interface

An advanced Computer Vision-based writing and drawing interface that utilizes Deep Learning for gesture recognition and spatial interaction.

## Key Features (Planned)

* **Z-Axis Depth Sensing:** Controlled line thickness based on hand distance.
* **Sketch-to-Art:** AI-powered refinement of air-sketched shapes.
* **Multimodal UI:** Voice-activated brush changes and gesture-based palette.

## Tech Stack

* **Language:** Python
* **Vision:** OpenCV, MediaPipe
* **Deep Learning:** TensorFlow/Keras (for custom gesture classification)
